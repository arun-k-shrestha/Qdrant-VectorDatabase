# Qdrant Learning Project

This repository is based on lessons from the official [Qdrant documentation](https://qdrant.tech/documentation).

## Goal
The purpose of this project is to **learn and teach how to use the Qdrant vector database**, including:

- Understanding the basics of **vector storage**  
- Building **semantic search** and **natural language search services**  
- Exploring **multi-vector search** and advanced retrieval concepts  

## Overview
This project serves as a hands-on learning resource to understand how **modern vector databases** power intelligent search, recommendation, and retrieval systems.
